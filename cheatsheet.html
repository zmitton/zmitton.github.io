Ruby Enumerables

  • each{|obj| block}

    Runs once for each item in enumerable. Has access to the current object in each iteration of the block. Does not change the origonal object.

    [3,4,5,6].each{|x| print x }
    #=>3 4 5 6

  • each_with_index{|obj, index| block}

    Does the same thing as "each" but allows for the extra variable "index" to be used within the block.

    [3,4,5].each_with_index{|x, i|
        print "Index #{i} = #{x} ; "
    }
    #=>Index 0 = 3 ; Index 1 = 4 ; Index 2 = 5 ;

  • inject(initial){|memo, obj| block}

    Takes the optional argument "initial" (default = 0), and a block. Uses the arument as first "memo", then sets memo to whatever the block returns on each iteration. Returns memo.

    • a = [5,6,7,8]
    puts a.inject(10){|memo, obj|
        memo += obj
    }
    #=> 36

  • reduce

    see inject

  • map{|obj| block}

    Returns a new array with the results of the block once for each element. If no block is given an enumerable is returned.

    (1..6).map{|num| num + 5} #=> [6,7,8,9,10,11]

  • collect

    see map

  • sort

    Returns an array with the items sorted.

    [3,5,4,7,6].sort #=>[3,4,5,6,7]
     [b,q,f,e,s].sort #=>[b,e,f,q,s]

  • include?

    Returns true if any member of enum equals ogj.

    [1,2,3,4,5].include? 5 #=> true
