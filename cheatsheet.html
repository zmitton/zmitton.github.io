Ruby Enumerables

  • each{|obj| block}

    Runs once for each item in enumerable. Has access to the current object in each iteration of the block. Does not change the origonal object.

    [3,4,5,6].each{|x| print x } #=>3 4 5 6

  • map{|obj| block}

    Returns a new array with the results of the block once for each element. If no block is given an enumerable is returned.


    (1..6).map{|num| num + 5} #=> [6,7,8,9,10,11]
